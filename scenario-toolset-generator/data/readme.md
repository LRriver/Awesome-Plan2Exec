# Data 原始数据

## 数据下载

运行以下命令下载原始数据：

```bash
wget https://www.modelscope.cn/datasets/nanbeige/ToolMind/resolve/master/graph_syn_datasets/graphsyn.jsonl
```

## 数据说明

### graphsyn.jsonl
ToolMind graphsyn原始数据集
- 记录数: 163,180
- 格式: `{"conversations": [...], "tools": [...]}`
- 来源: [ToolMind](https://www.modelscope.cn/datasets/nanbeige/ToolMind)
